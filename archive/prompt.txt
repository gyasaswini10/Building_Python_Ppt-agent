## 🛠️ **Tool Functions Reference**

### **Agent Tools (`agent_ppt.py`)**

#### **Core Functions:**
```python
# Main agent class
agent = AutoPPTAgent(user_request, output_file)

# Execute presentation creation
await agent.execute()
```

#### **API Management:**
```python
# Check available API keys
agent._get_available_openrouter_keys()
agent._get_available_hf_tokens()

# Handle API failures
agent._try_openrouter(prompt)
agent._try_huggingface(prompt)
```

#### **Content Processing:**
```python
# Extract presentation topic
agent.extract_topic(request)

# Validate bullet points
agent._llm_bullet_ok(bullet_text)

# Match content to slide titles
agent.bullet_matches_slide(topic, title, bullet)
```

### **PowerPoint Tools (`ppt_mcp_server.py`)**

#### **Available MCP Tools:**
```python
# Create new presentation
create_pptx(title="My Presentation")
# Returns: {"ok": True, "session_id": "uuid"}

# Add slide with content
add_slide(session_id="uuid", slide_title="Title", bullets=["point1", "point2"])
# Returns: {"ok": True, "slide_index": 1}

# Delete slide by index
delete_slide(session_id="uuid", slide_index=2)
# Returns: {"ok": True, "slides_total": 5, "deleted_index": 2}

# Get slide information
get_slide_info(session_id="uuid", slide_index=-1)
# Returns: {"ok": True, "slides_info": [...], "total_slides": 6}

# Save presentation
save_presentation(session_id="uuid", output_path="path/to/file.pptx")
# Returns: {"ok": True, "output_path": "path/to/file.pptx"}
```

#### **Styling Features:**
- Dark theme with navy background (#0F192D)
- Gold accent text (#FFDF80)
- Professional font sizing and positioning
- Automatic text wrapping and formatting

### **Research Tools (`research_mcp_server.py`)**

#### **Available MCP Tools:**
```python
# Search Wikipedia for content
search_topic(query="solar energy", slide_title="Introduction")
# Returns: {"ok": True, "points": [...], "thumbnail": "url", "source": "wikipedia"}

# Get word definitions
get_definition(word="photosynthesis")
# Returns: {"ok": True, "definition": "process...", "source": "dictionary"}
```

#### **Content Processing:**
```python
# Extract keywords from text
extract_keywords("Solar energy is renewable energy from sunlight")

# Clean bullet points
clean_bullet_point("• this is a bullet point")
# Returns: "This is a bullet point."

# Validate content quality
validate_bullet_point("Good quality educational content")
# Returns: True/False
```

## 🧪 **Troubleshooting Guide**

### **Common Issues:**

#### **API Key Problems:**
```bash
# Check if API keys are loaded
python -c "
import os
from dotenv import load_dotenv
load_dotenv()
print('OpenRouter keys:', bool(os.getenv('OPENROUTER_API_KEY_1')))
print('HF tokens:', bool(os.getenv('HF_TOKENS')))
"
```

#### **Server Connection Issues:**
```bash
# Test if MCP servers are running
python -c "
import asyncio
import sys
from mcp import ClientSession, StdioServerParameters
from mcp.client.stdio import stdio_client

async def test_servers():
    try:
        ppt_params = StdioServerParameters(command='python', args=['client/Modular/code/ppt_mcp_server.py'])
        async with stdio_client(ppt_params) as (read, write):
            await asyncio.sleep(1)
            print('✅ PowerPoint server reachable')
    except Exception as e:
        print(f'❌ PowerPoint server error: {e}')

asyncio.run(test_servers())
"
```

#### **File Permission Issues:**
```bash
# Test output directory permissions
python -c "
from pathlib import Path
test_file = Path('savingfolder_output/test.txt')
try:
    test_file.touch()
    print('✅ Directory writable')
    test_file.unlink()
except Exception as e:
    print(f'❌ Permission error: {e}')
"
```
