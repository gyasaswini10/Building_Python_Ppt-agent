# Video Demo Script (2 minutes)

## Goal
Show the agent creating a `.pptx` from a single-sentence prompt without mouse interaction.

## Suggested Timeline (Approx.)

1. 0:00–0:15: Introduce the prompt and what the agent will generate.
2. 0:15–0:45: Run the CLI command (`python agent_ppt.py "..."`) or run the notebook cell.
3. 0:45–1:15: Show the resulting `.pptx` opening and quickly scroll through slides.
4. 1:15–1:45: Mention that it plans first, then writes slides via MCP tools (no hardcoded PPT content).
5. 1:45–2:00: Close with a second quick prompt (optional) or a robustness example (Claude/search failure fallback).

## What to Mention (Grader Keywords)
- Agentic planning step before slide writing
- MCP usage (PowerPoint writer + research)
- Error handling / fallback behavior
